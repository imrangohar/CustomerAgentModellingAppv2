# Quick Start

Use this checklist to complete one end-to-end onboarding cycle.

## 1) Start the app
1. Open terminal in project root:
   - `/Users/akale/Projects/my-codex-webapp`
2. Install dependencies:
   - `npm install`
3. Run dev server:
   - `npm run dev`
4. Open the `Local:` URL from terminal, then go to `/onboarding`.

## 2) Complete one representative workflow

### Step 1: Onboarding Scope & Setup
- Fill:
  - `Company Name`
  - `Rollout Track`
  - `ERP/P2P System`
  - `PO Usage Band`
  - `AP Team Size`
  - `Industry Vertical`
- Click `Apply Tailoring`.

### Step 2: Policy Inputs & Prefill
- Click `Demo Mode: Auto-load Sample Policy Pack` for a quick seeded run.
- Review suggested rows in:
  - `Controller Priority Review`
  - `Prefill Review Queue`
- Click `Accept` on valid suggestions.

### Steps 3 to 8: Answer and confirm
- Open each step and answer required questions.
- Click `Confirm` for accepted values.
- Use `Assign` if information is missing.

### Step 9: Review, export, activate
- Check section statuses and counts.
- Click `Export Policy and Controls` (PDF).
- Click `Activate` to open `/inbox` (`Status`).

## 3) Validate collaboration flow (optional)
1. Assign one question.
2. Open `/collaboration`.
3. Open responder link and submit a response.
4. Click `Accept` in collaboration dashboard.

## 4) SOP output
Go to `/sop` and test:
- `Save Version`
- `Download PDF`
- `Export JSON`
